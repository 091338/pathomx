Annotate

Add external class and label annotations to your dataset
Martin A. Fitzpatrick

Introduction

Some data input sources do not allow inclusion of sample classes or labels required for subsequent analysis. Using this plugin you can load both types of data from external files (.csv) and map them onto an existing internal dataset.

Quick start

Select source data and then open an annotation mapping file. This plugin will identify matching dimensions on which to map the data and present you with the resulting options. Select the correct mapping for your purposes and save.

The annotated data will be available to further analysis.
