Introduction

PLS_Toolbox software is the world’s most extensive suite of essential and advanced chemometric multivariate analysis tools for use within the MATLAB® computational environment.

To use PLS_Toolbox you need a license available here. Download and install PLS_Toolbox in MATLAB before using this plugin.

This plugin is a wrapper interface to a selection of the tools offered by PLS_Toolbox. Use of this plugin requires an installation and license for Matlab and PLS_Toolbox. You can download PLS_Toolbox from here.
