Pathway connects

Visualise inter-pathway connections by reaction, metabolite or compartment
Martin A. Fitzpatrick

Introduction

Metabolic pathways are conceptually useful for understanding the implications of metabolic change. This simple plugin helps to keep these changes in context, by visualising the links between identified changed pathways at a reaction or metabolite level. Two highly regulated pathways that are highly connected may reflect feedback.

Quick start

Select source pathway list and a pathway connection map will be drawn for the pathways. You can manually provide a list of pathways also. If you have metabolite change change data this can optionally be overlaid on the pathway connection map. @end
